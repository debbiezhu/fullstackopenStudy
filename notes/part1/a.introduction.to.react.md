**vite**: 一个 尤雨溪 的作品. 相当于 uvicron 之于 fastapi. 是一个快速启动的工具.

安装 `vite`: `npm create vite@latest part1 -- --template react`

安装 `part1`: `cd part1; npm install`

运行 `part1`: `npm run dev`

`vite` 启动默认的端口是 `5173`; 否则, 从 `5173` 递增.
